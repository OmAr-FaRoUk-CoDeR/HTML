خصائص أشرطة التمرير في CSS


خصائص أشرطة التمرير في CSS تحدد التأثيرات المختلفة على شريط التمرير الجانبي للصفحة.


أشرطة التمرير في CSS :
خصائص أشرطة التمرير تسمح لك بالتحكم في شريط التمرير بالصفحة ، كإعداد لون القاعدة و لون الخلفية و لون السهم و لون الواجهة أو لون الظل كما بمكنك إضافة تأثير ثلاثي للشريط.


خصائص أشرطة التمرير :

دعم المتصفحات :
NS: Netscape
IE: Internet Explorer
W3C: Web Standard

الخاصية الوصف القيمة NS IE W3C
scrollbar-base-color إعداد لون القاعدة لأشرطة التمرير الطولية و العرضية. scrollbar-base-color 6.0 4.0 CSS1
scrollbar-arrow-color إعداد لون السهم لأشرطة الطولية و العرضية. scrollbar-arrow-color 6.0 4.0 CSS1
scrollbar-track-color إعداد لون الخلفية لأشرطة التمرير الطولية و العرضية. scrollbar-track-color 4.0 4.0 CSS1
scrollbar-face-color إعداد صورة كخلفية. scrollbar-face-color 4.0 4.0 CSS1
scrollbar-highlight-color إعداد الإضاءة لأشرطة التمرير الطولية و العرضية. scrollbar-highlight-color CSS1
scrollbar-3dilight-color إعداد تأثير ثلاثي لأشرطة التمرير الطولية و العرضية. scrollbar-3dilight-color 4.0 4.0 CSS1
scrollbar-darkshadow-color إعداد لون الظل لأشرطة التمرير الطولية و العرضية. scrollbar-darkshadow-color 4.0 4.0 CSS1
scrollbar-shadow-color إعداد لون الظل لأشرطة التمرير الطولية و العرضية أيضاً. scrollbar-shadow-color 4.0 4.0 CSS1
